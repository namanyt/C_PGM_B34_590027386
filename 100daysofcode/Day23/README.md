## How to Compile & Run

1. Open a terminal and navigate to this folder.
2. Compile the program using `gcc`:
   ```bash
   gcc 45_series_sum.c -o 45_series_sum
   ./45_series_sum
   ```
3. Repeat for other programs, replacing the filenames accordingly.
